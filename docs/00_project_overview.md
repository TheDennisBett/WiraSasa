# WiraSasa Project Overview

## Product
WiraSasa is a two-sided home services marketplace. A single account can operate as a client or provider, which drives the need for shared identity, role switching, and a navigation model that supports both service booking and job fulfillment.

## Phase 1 Scope
- Bootstrap a runnable Flutter application
- Establish a scalable folder structure
- Define global theming and visual direction
- Implement the navigation shell and placeholder entry points for all major flows
- Prepare project documentation for later phases

## Current Status
Phase 1 is implemented with mock data, placeholder screens, Riverpod state, and centralized routing. Real authentication, maps, APIs, persistence, and job lifecycle logic are deferred to subsequent phases.
